# Sistema de Gestión de Personal de Seguridad

Aplicación web desarrollada con PHP y MySQL bajo el patrón de arquitectura MVC. Permite la gestión administrativa de guardias de seguridad, incluyendo control de roles y remuneraciones.

## 📋 Características
- **CRUD Completo:** Crear, Leer, Actualizar y Eliminar registros.
- **Arquitectura MVC:** Separación lógica en Modelos, Vistas y Controladores.
- **Validaciones:** Control de datos tanto en cliente (JS) como en servidor (PHP).
- **Base de Datos:** MySQL.
